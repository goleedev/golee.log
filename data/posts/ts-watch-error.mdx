# 🚧 TypeScript 환경 설정 해결하기

타입스크립트를 **글로벌로 설치 후 특정 모듈을 사용하려면 에러**가 발생합니다.

```bash

Err: Cannot find module 'typescript/bin/tsc'
error Command failed with exit code 9

```

이 모듈을 계속해서 시행 하려면 **타입스크립트를 글로벌이 아닌 로컬**로 설치해주어야 합니다.

```bash

yarn global remove typescript

yarn add typescript

```
